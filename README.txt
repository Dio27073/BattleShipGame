Java Battleship Game

Description:
This project is a Java-based implementation of the classic two-player game Battleship. The objective of the game is for each player to try and sink the others hidden ships.
This implementation includes a graphical user interface, turn-based gameplay, and follows standard Battleship rules.

Team Members:
Member 1: Claudio Olmeda Florio
Member 2: Josiah Ericksen

Features:
- Graphical User Interface (GUI) for interactive gameplay.
- Support for two human players.
- Turn-based mechanics, ensuring each player takes turns without seeing the opponent's board.

Rules of The Game:
The game follows standard Battleship rules. Players take turns to guess the location of the opponents ships on a grid. First to sink all ships wins. 

Setup and Installation:
- Java SE 17 SDK
- Run the JAR file using the command: java -jar hwx.jar

Game Instructions:
1. Start the game by running the JAR file.
2. Ships are automatically placed randomly.
3. Players take turns guessing the location of the opponent's ships.
4. The game ends when all ships of a player are sunk.
